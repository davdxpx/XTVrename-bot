# Changelog

## [Unreleased] - 2026-04-15

This release introduces a new YouTube Tool that lets users download videos, audio tracks, subtitles, thumbnails, and metadata directly through the bot, dramatically expanding what XTV MediaStudio can pull from the web. The File Converter has been upgraded to a "mega edition" with broader format coverage, and its UI — along with the new YouTube Tool UX — has been polished for clearer navigation. The dumb-channel add flow has been rewritten as a unified three-step wizard, making it easier for admins to register new auto-forward channels without memorizing separate commands. Together these changes focus on making media ingestion and conversion workflows more self-service and less error-prone.
